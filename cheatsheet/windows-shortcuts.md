# Windows ショートカット チートシート

## ウィンドウ管理
- `Win+←/→/↑/↓`：分割・最大化・最小化
- `Win+Shift+←/→`：別モニターへ移動
- `Alt+Tab`：直前ウィンドウへ
- `Alt+Shift+Tab`：逆方向
- `Win+D`：デスクトップ表示
- `Win+Tab`：仮想デスクトップ

## 起動・切替
- `Win+数字`：タスクバーN番目を起動/切替（**1〜5に常用アプリ固定**）
- `Win`：スタートメニュー（そのままアプリ名検索）
- `Win+E`：エクスプローラ
- `Win+L`：ロック

## 入力・コピペ
- `Win+V`：クリップボード履歴（**要設定で有効化**）
- `Win+.`：絵文字・記号
- `Win+Shift+S`：範囲スクリーンショット

## 検索
- `Win+S`：検索バー
